Here is a small description about few of our MS's:

ask-leni : contains APIs for supporting MQL generation (query to MQL transformation), Question Completion, Flow etc which are major features/components related to ASK

auto-query-completion: APIs for supporting spellcheck and OOC feature, Guided Ask etc

ontology ms :  All the APIs related to ontology ( As of now semantic model has an ontology in the backend. In future, this will be removed and it will be part of the semantic model pipeline itself)

config ms: All the APIs for supporting solution configurations like default measure/date etc

utility & question generation: APIs for supporting multi-tile/multi-semantic model

narratives-for-leni: API for Narrative generation for different variants of questions

config-development: APIs for handling Jinja templates for narratives

lxai: APIs supporting explainable AI feature


-----------------------------UPDATING Ollama------------------------------
cd /usr/bin
sudo rm -rf ollama
sudo curl -L https://ollama.com/download/ollama-linux-amd64 -o /usr/bin/ollama
sudo chmod +x /usr/bin/ollama

sudo systemctl daemon-reload
sudo systemctl enable ollama

sudo systemctl start ollama
